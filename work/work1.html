MY PORTFOLIO

Design / Coding / Deploy

担当内容

企画 / デザイン / コーディング / デプロイ

使用ツール・言語

HTML / CSS / JavaScript

制作期間

1ヶ月

概要

私、荒井愛理の制作実績やスキル、経歴を紹介するためのポートフォリオサイトです。企画からデザイン、コーディング、サーバーへのデプロイまで、すべて自分自身で行っています。

デザイン

クリーンで信頼感のある印象を与えるため、クリーム色（#fdfcf7）をベースに、アクセントカラーとしてティール（#7db8b3）を使用しました。フォントは全体をクリーンなサンセリフ体で統一し、情報の見やすさを重視しています。

工夫した点

PCでもスマートフォンでも快適に閲覧できるよう、CSS GridやFlexboxを用いたレスポンシブデザインに対応しています。

特に「Skill」セクションのカードレイアウトでは、`display: flex` と `flex-direction: column` を駆使し、カードの高さが3枚で揃いつつも、中身のテキストが（`overflow-y: auto` で）はみ出さない堅牢なレイアウトを実装しました。

また、ハンバーガーメニューは `common.js` で制御し、CSS Transform を使ったスムーズなスライドアニメーションを実装しています。

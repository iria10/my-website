画面デザイン / データ管理

画面デザイン / データ管理 / データ収集

使用ツール・言語

PHP / SQLite / JavaScript / OpenAI Vision API / Leaflet.js

制作期間

4ヶ月 (大学ゼミ)

制作リンク

TRASH NAVIGATION_GitHub

概要

訪日外国人観光客向けのゴミ分別支援システムです。言語の壁や文化の違いによってゴミ分別に困る観光客をターゲットに、画像認識(OpenAI Vision)でゴミの種類を判別し、現在地から中野区内の最寄りのゴミ箱を地図(Leaflet.js)上に表示するWebアプリケーションです。

背景と私の役割

大学のゼミにて、4か月間にわたり企画・開発しました。私はチーム内で画面デザイン、データ管理の二つの役割を担当しました。

画面デザインでは、日本語・英語・中国語など複数言語に対応したメニュー機能とシステム全体の見た目を設計し、誰でも直感的に操作できる画面を実現しました。
また、このシステムをスマートフォンで使用できるようにするため、画面デザインの対応にも意識を向けました。

データ管理では、ゴミ箱の位置情報を登録・保存する仕組みを設計し、中野区内のゴミ箱データの収集・整理を実施してシステムの実用性を高めました。

機能

  • 画像アップロード機能
  • AIによるゴミの種類判別 (燃えるゴミ、瓶、缶など)
  • 位置情報に基づいた最寄りゴミ箱検索 (中野区内)
  • 地図 (OpenStreetMap + Leaflet.js) 上への結果表示
  • 多言語対応 (日本語, English, 简体中文)
  • ゴミ箱データの登録・閲覧機能
